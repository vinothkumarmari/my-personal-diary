# 📔 My Personal Diary

A beautiful, private digital diary with journal entries, book reading with PDF support, and a drawing pad.

![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Features

### 📔 Journal Entries
- Write daily thoughts and reflections
- Mood tracking with emojis (😊😔🤩😌😐😰)
- Search through all entries
- Date & time stamps automatic

### 📚 Book Reader
- Import PDF books from your device
- Realistic open book UI with page turning animation
- 8 beautiful reading fonts (Merriweather, Crimson Text, Libre Baskerville, etc.)
- Adjustable font size (12px - 28px)
- Reading progress saved automatically
- Personal book library

### 🎨 Drawing Pad
- Free drawing canvas
- Multiple colors + custom color picker
- Adjustable brush size
- Eraser tool
- Save drawings to library
- Download as PNG
- Touch/stylus support

### 🔐 Privacy
- Password protected
- All data stored locally in YOUR browser
- No cloud, no server storage
- Completely private and secure

### 🌙 Dark/Light Mode
- Toggle between themes anytime
- Comfortable reading day or night

## 🚀 Quick Start

### Run Locally

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/my-personal-diary.git

# Navigate to project
cd my-personal-diary

# Install dependencies
npm install

# Start server
npm start

# Open http://localhost:3000
```

### Or Just Open the HTML File
Simply double-click `index.html` to open in your browser!

## 🌐 Deploy to Render

1. Push this repo to GitHub
2. Go to [Render.com](https://render.com)
3. Create New → Web Service
4. Connect your GitHub repo
5. Settings:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
6. Deploy!

## 📝 Usage

1. **First Time:** Create a password (min 4 characters)
2. **Write:** Click "Write" tab to add journal entries
3. **Read:** Import PDF books in "Books" tab
4. **Draw:** Create artwork in "Draw" tab
5. **Search:** Find old entries by keyword

## ⚠️ Important Notes

- **Remember your password** - no recovery option!
- **Browser storage** - clearing cache deletes data
- **Backup regularly** - export important entries

## 🛠️ Tech Stack

- React 18
- PDF.js for book reading
- HTML5 Canvas for drawing
- Express.js for hosting
- LocalStorage for data persistence
